# 情報源ポリシー（source-policy）

## 優先順位

1. 法令・公的機関（法務省・国税庁・厚生労働省・裁判所・消費者庁・自治体・JPRS等の一次情報）
2. 専門団体（司法書士会・弁護士会・終活カウンセラー協会等）
3. 運営者の実体験・相談経験（体験談として明示できるもののみ）
4. 商用メディア（方向性の参考のみ。数値・制度の根拠にしない）

## ルール

- 法制度・期限・金額を本文に書くときは、一次情報URL＋確認日を記事内または本ファイル配下に記録する
- 「原則」「一般に」を付け、個別事情は専門家へ誘導する
- 確認できない情報は書かない（推測禁止）
- 体験談は運営者から提供された事実のみ。生成・脚色禁止
- 更新日（dateModified）は実質更新時のみ変更
